INTRODUCTION

In many webpages the loading time is affected by the nature of information they contain (large images and multimedia in general); visitors often abandon them before the loading ends, unaware of the webpage’s loading status. To avoid this situation, we developed the Loading WebPage extension.

Loading WebPage is an extension for Adobe Dreamweaver MX or later. It shows a screen that indicates the status of webpage loading and hides the screen when the loading process is over.

^ Top

Requirements

Adobe Dreamweaver MX or later.

Installation

To install Loading WebPage, simply execute the file TE_LoadingPage.mxp or use Adobe Extension Manager software.

Once installed, the extension can be accessed via the menu option “Insert > Loading WebPage”, or via “InsertBar > Common > Insert Loading WebPage” ( ).

use

To use Loading WebPage extension, the page where it will be inserted needs to be opened and saved. When you insert a Loading WebPage, the following setup window will be shown:

The option Insert Mask allows to place a mask on the entire webpage that hides and protects the content of the page until it is totally loaded.

Note 1: for the Loading WebPage to be most efficient as far as loading time is concerned, it should be inserted immediately after the opening tag “BODY”.

Note 2: If you wish to personalize mask colours or other elements of the loading status screen, you simply need to make changes to the styles file Loading.css, which is automatically created when you insert a Loading WebPage.

Conclusion

Developing web-based applications or internet sites, it is fundamental to guide and inform users so that they do not feel disoriented; this is why in webpages where loading is not immediate, Loading WebPage extension can come extremely handy and even prevent users from premature leaving.
